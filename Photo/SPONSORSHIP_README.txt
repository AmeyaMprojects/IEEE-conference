SPONSORSHIP OPPORTUNITY PDF SETUP
===================================

"sponsorship-opportunity.pdf"

"Sponsorship Opportunity" button in the header navigation.

Location: Photo/sponsorship-opportunity.pdf

If you need to change the PDF file name or location, update the link in:
index.html (line with sponsorship-btn)
